# 🏥 Astha Paramedical College

A client based modern and responsive landing page developed for **Astha Paramedical College** to showcase courses, campus facilities, admissions, and provide an online appointment experience.

## ✨ Features

- 🏥 Modern landing page
- 📚 Courses & admissions section
- 🧪 Infrastructure showcase
- 📅 Appointment booking form
- 📞 Contact & enquiry section
- 📱 Fully responsive design

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS

## 🚀 Run Locally

```bash
npm install
npm run dev
```
